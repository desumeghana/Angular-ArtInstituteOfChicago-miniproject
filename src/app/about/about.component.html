Home Page

Home Page gives a Welcome message and a navbar The navbar leads to various components as required by the user

Artworks Page

Artworks Page gives a list of all the arts from the API data and also links the data according to the paginator It also have a search functionality which filters the artworks acccording to the search input

Wishlist Page

Wishlist Page displays the all wishlisted items which gets added to the page and we can also remove the artwork from the wishlist by deselecting the favorite icon again & whenever the wishlist is empty it displays as empty

View Page

View page is a detailed view of a particular artwork. It gets initialized whenever you click on the required image of the artwork

Share Feature

Share is a feature where you can share the particular artwork to different social media applications
